# Telegram Bot

## Инструкция
1. Создайте бота у [@BotFather](https://t.me/BotFather), получите токен.
2. Установите переменную окружения `BOT_TOKEN`.
3. Установите зависимости: `pip install -r requirements.txt`
4. Запустите бота: `python main.py`
